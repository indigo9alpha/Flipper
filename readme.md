# flipper

flipperはシンプルな読書管理アプリです。

読んだページの数がアニメーションで表示され、積み重ねを視覚的に実感できます。


## 使い方

ISBNコードを入力し、本を登録します。


## 使用技術

- React
- redux
- redux-thunk
- react-router
- connected-react-router
- react-countup
- Firebase
- Firebase Authentication
- Cloud Firestore
- Firebase Hosting
- Material-UI
- Google Books API
